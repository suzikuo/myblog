思路：

1、通过selenium截取整个页面

2、使用Image截取每个字体，

3、使用图片识别技术（推荐百度云文字识别）

到此，全部的字体已经截图完成。接下来将进行文字识别。
使用到的是调用百度文职识别api接口，

https://ai.baidu.com
登录百度账号，进去控制台，选择文字识别，创建应用，并复制三个参数

最后，再使用高级识别接口（client.basicAccurate(img），进行第二次识别，解决上一步识别失败的图片，剩余的再手动映射吧。
