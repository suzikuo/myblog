Batch 可以帮助我们在 AWS 云上运行批量计算工作负载。

2.3 创建任务定义


任务类型我们选择单节点

输入名称与超时时间

选择使用Fargate作为运行环境，并开启分配公共IP。

进入ECR，选择我们刚才创建的镜像仓库，复制其ARN。

返回创建，输入映像与命令
命令：python backtest.py 【数据源所在S3桶名】 【要回测的源数据文件名】 【结果存储S3桶名】
